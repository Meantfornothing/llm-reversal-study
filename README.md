# LLM Reversal Study: Diffusion vs. Autoregression

This research project explores the "Reversal Curse" in Large Language Models (LLMs) through two lenses:
1. **Quantitative:** A benchmark comparing 4-bit quantized Autoregressive (Llama-3.2-3B) and Diffusion (LLaDA-8B) models.
2. **Qualitative:** Semi-structured interviews with Human-centered AI students regarding their mental models of language generation.

## Setup
1. `pip install -r requirements.txt`
2. Run `quantitative/benchmark.ipynb` for model testing.
3. Reference `qualitative/protocols/` for interview structures.
